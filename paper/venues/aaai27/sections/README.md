# AAAI 2027 Local Sections

This directory now carries the AAAI-local main-body version of the
figure-driven VLM/InternNav story. The files were synchronized from the ACL
local sections so the AAAI candidate PDF does not fall back to the older
tool-first shared scaffold.

Current local sections:

- `abstract.tex`: frames material conversion as a controlled perturbation for VLM grounding.
- `intro.tex`: foregrounds the language-grounding and embodied-data reliability story.
- `related.tex`: covers grounding, VLN, embodied simulation, rendering shifts, and proxy metrics.
- `method.tex`: organizes the protocol around material perturbation and evidence gates.
- `results.tex`: reports VLM grounding, material-effect boundaries, and official InternNav sanity evidence.
- `discussion.tex`: keeps embodied and performance claims scoped.
- `conclusion.tex`: closes on VLM and embodied evidence boundaries.
- `limitations.tex`: includes selected InternNav path panels and scoped limitations.
- `ethical-considerations.tex`: records broader-impact and data-use notes.

Shared tables, figures, bibliography, evidence manifests, and appendix assets
remain under `../../shared/`.
