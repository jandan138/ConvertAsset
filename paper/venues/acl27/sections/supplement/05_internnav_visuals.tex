\section{InternNav / DualVLN Navigation Visual Supplement}
\label{sec:supp-internnav-visuals}

This appendix expands the selected navigation evidence. The quantitative result
remains tied to the 99-episode official KuJiaLe paired run. The panels here are
orientation evidence: they show start, mid, or end frames together with
top-down trajectory overlays so that a reader can inspect what the selected
videos look like.

\subsection{Media Inventory Scope}

The selected video material can be uploaded as a separate media bundle only
after final author approval. The PDF supplement uses still panels
instead of raw frame directories. The intended media package contains compressed
clips, stills, a manifest mapping each clip to case id and metrics, and a short
README stating that the videos are selected qualitative evidence. The default
safe review packet excludes videos until this author/legal approval step is explicitly
approved.
