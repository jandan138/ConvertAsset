# Superpowers Process Artifacts

This directory is an internal process exception to the main documentation taxonomy.

ConvertAsset's first-class documentation uses:

- `docs/design/`
- `docs/operations/`
- `docs/records/`
- `docs/reference/`
- `archive/`

`docs/superpowers/` stores design specs and implementation plans produced by the
superpowers workflow so later agents can audit the decisions that led to a
large documentation or implementation change.
