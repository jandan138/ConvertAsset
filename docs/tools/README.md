# Tools & Utilities

This directory contains documentation for various utility scripts and helper tools in the ConvertAsset project.

## Available Tools

### [Material Inspector](inspect_material.md)
Documentation for the material inspection tool, which helps debug and analyze material properties in USD files.

### [Isaac Python Wrapper](isaac_python_wrapper.md)
Details about the wrapper used to interface with Isaac Sim's Python environment, handling environment variables and path setups.

## Usage
Most tools are located in the `tools/` or `convert_asset/tools/` directory in the source code. Refer to individual documents for specific usage instructions.
