distribution functions 与 material struct

MDL 的物理接口围绕三类distribution function（分布函数）组织：描述表面散射的 BSDF（Bidirectional Scattering Distribution Function）、描述自发光的 EDF（Emissive Distribution Function）、以及描述体积散射的 VDF（Volume Distribution Function）。这一节从这三类函数出发，理解 MDL 材质的物理语义，以及它们为何比 Cook-Torrance 参数更"底层"。

BSDF：表面散射分布

BSDF 是 MDL 材质最核心的部分，它直接对应渲染方程里的 $f_r$。MDL 提供了一套基元 BSDF（elemental BSDFs）作为构建块，常用的包括：

  • diffuse_reflection_bsdf——基于 Oren-Nayar 模型的漫反射（也涵盖纯 Lambertian）。
  • diffuse_transmission_bsdf——Lambertian 透射分量。
  • specular_bsdf——完全镜面反射或折射（控制 IOR），用于玻璃、镜面。
  • simple_glossy_bsdf——光泽反射/透射（参数化的粗糙度）。
  • backscattering_glossy_reflection_bsdf——支持后向散射的光泽反射，常用于布料。
  • measured_bsdf——基于测量数据的各向异性 BSDF（BTF 文件）。

这些基元之间可以通过组合算子（combiners）叠加，形成复杂的多层材质：

layered_material.mdl
// 用 weighted_layer 把漫反射底层 + 光泽顶层叠加
// Fresnel 项由 fresnel_layer 自动计算
material layered_metal(
    color base = color(0.8, 0.6, 0.2),
    float roughness = 0.2,
    float ior_val = 1.5
) = material(
    surface: material_surface(
        scattering: df::fresnel_layer(
            ior: ior_val,
            layer: df::simple_glossy_bsdf(
                roughness_u: roughness,
                tint: color(1.0)
            ),
            base: df::diffuse_reflection_bsdf(tint: base)
        )
    )
);
组合算子 MDL 提供 weighted_layer（权重混合）、fresnel_layer（Fresnel 自动权重）、custom_curve_layer（自定义曲线权重）、以及 color_custom_curve_layer。这让复杂的分层材质（如带涂层的车漆）可以用组合方式表达，而不是硬编码一个着色器。

EDF 与 VDF：发光与体积

EDF（发光分布函数）统一了光源与发光材质的表达——MDL 里没有特殊的"光源类型"，光源就是发光的几何体：

  • diffuse_edf——各方向均匀发光（Lambertian 光源）。
  • spot_edf——余弦分布的点状发光（聚光灯）。
  • measured_edf——从 IES 光度测量文件读取的任意光分布。

VDF（体积分布函数）描述光在体积内部的散射与吸收，与 BSDF 配合可以表达玉石、皮肤（SSS）、烟雾等效果：

volume_material.mdl
// 有吸收 + 各向异性散射的体积材质
material jade_like() = material(
    surface: material_surface(
        scattering: df::specular_bsdf(mode: scatter_transmit)
    ),
    ior: color(1.5),
    volume: material_volume(
        scattering: df::anisotropic_vdf(directional_bias: 0.2),
        absorption_coefficient: color(0.02, 0.4, 0.3),
        scattering_coefficient: color(0.5, 0.3, 0.1)
    )
);

与 UsdPreviewSurface 的对比

学完 MDL 的 distribution function 体系，回头看 UsdPreviewSurface 就能理解它的位置：UsdPreviewSurface 是一个高层参数化封装——它把底层的 Cook-Torrance specular、Lambertian diffuse、Fresnel 等组合成一个固定的 metallic-roughness 接口，更易于艺术家使用，但灵活性远不如 MDL 的 BSDF 组合体系。

小结 MDL 的 BSDF / EDF / VDF 体系是物理材质的"完备语言"；UsdPreviewSurface 是这个空间里一个参数化的点。no-MDL 转换的难点正是：从完备语言中的一个任意点，近似到那个固定的参数化封装——不可避免会有信息损失。
