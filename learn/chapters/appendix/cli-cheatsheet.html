CLI cheat-sheet

所有命令都通过 ./scripts/isaac_python.sh ./main.py <subcommand> [args] 调用。isaac_python.sh 负责设置 Isaac Sim 的 Python 环境，然后执行 main.py。默认子命令（无子命令时）为 no-mdl。

no-mdl — MDL → UsdPreviewSurface

递归处理 USD 文件树，为每个文件生成 *_noMDL.usd 兄弟文件，材质网络替换为 UsdPreviewSurface。

no-mdl
# 基本用法
./scripts/isaac_python.sh ./main.py no-mdl /path/to/scene.usd

# 只写 noMDL USD，跳过 summary / audit 文件
./scripts/isaac_python.sh ./main.py no-mdl /path/to/scene.usd --only-new-usd

# 纹理路径写为绝对路径（legacy 行为）
./scripts/isaac_python.sh ./main.py no-mdl /path/to/scene.usd \
    --resolve-textures-to-absolute

mesh-faces — 统计网格面数

mesh-faces
./scripts/isaac_python.sh ./main.py mesh-faces /path/to/scene.usd
# 输出：Total render mesh faces: 847392

mesh-simplify — QEM 网格简化

mesh-simplify
# 干跑（不写文件），查看简化效果
./scripts/isaac_python.sh ./main.py mesh-simplify \
    /path/to/scene.usd --ratio 0.5 --backend py

# 实际简化并写出
./scripts/isaac_python.sh ./main.py mesh-simplify \
    /path/to/scene.usd \
    --ratio 0.5 \
    --backend py \
    --apply \
    --out /path/to/scene_simplified.usd \
    --progress

# 用目标面数代替比例
./scripts/isaac_python.sh ./main.py mesh-simplify \
    /path/to/scene.usd \
    --target-faces 100000 \
    --backend py --apply --out /path/to/out.usd

# C++ pybind11 backend（需先编译）
./scripts/isaac_python.sh ./main.py mesh-simplify \
    /path/to/scene.usd --ratio 0.5 --backend cpp-uv \
    --apply --out /path/to/out.usd
backend 选项 py：纯 Python，无需编译，适合快速上手。
cpp：调用原生 C++ 可执行文件（native/meshqem/build/meshqem）。
cpp-uv：pybind11 in-process C++ backend，性能最佳，UV 保持最准，需先编译 native/meshqem。

inspect — 检查材质网络

inspect
# 查看 MDL 着色器网络
./scripts/isaac_python.sh ./main.py inspect \
    /path/to/scene.usd mdl /Looks/MaterialName

# 查看 UsdPreviewSurface 网络
./scripts/isaac_python.sh ./main.py inspect \
    /path/to/scene_noMDL.usd usdpreview /Looks/MaterialName

export-glb — USD → GLB

export-glb
# 推荐输入：*_noMDL.usd（MDL 已转换）
./scripts/isaac_python.sh ./main.py export-glb \
    /path/to/scene_noMDL.usd \
    --out /path/to/output.glb

usd-to-glb — 一步到位流水线

自动执行：no-mdl → 生成中间 *_noMDL.usd → export-glb → 删除中间文件。

usd-to-glb
./scripts/isaac_python.sh ./main.py usd-to-glb \
    /path/to/scene.usd \
    --out /path/to/output.glb

# 保留中间 *_noMDL.usd 文件
./scripts/isaac_python.sh ./main.py usd-to-glb \
    /path/to/scene.usd \
    --out /path/to/output.glb \
    --keep-intermediate

render-single — 本地 Isaac Sim 出图

render-single
# 4 视角 512×512 默认出图
./scripts/isaac_python.sh ./main.py render-single \
    /path/to/asset.usd \
    --out /path/to/renders

# 完整参数示例
./scripts/isaac_python.sh ./main.py render-single \
    /path/to/asset.usd \
    --out /path/to/renders \
    --width 1024 --height 1024 \
    --views 4 \
    --elevation 35 \
    --azimuth-offset 0 \
    --focal-mm 18 \
    --warmup-steps 100 \
    --render-steps 16 \
    --renderer PathTracing \
    --naming-style view \
    --background-color 40,40,40 \
    --overwrite

thumbnails — 场景实例批量出图

thumbnails
./scripts/isaac_python.sh ./main.py thumbnails \
    /path/to/scene.usd \
    --out /path/to/thumbnails \
    --width 600 --height 450 \
    --views 6 \
    --warmup-steps 1000 \
    --render-steps 8 \
    --focal-mm 9

export-mdl-materials — 导出 MDL 材质

export-mdl-materials
# 以 UsdPreviewSurface 模式导出（附球形预览场景）
./scripts/isaac_python.sh ./main.py export-mdl-materials \
    /path/to/scene.usd \
    --out-dir-name exported_materials \
    --mode preview \
    --emit-ball
