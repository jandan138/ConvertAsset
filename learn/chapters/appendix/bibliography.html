Bibliography 参考资料

本页按主题收录全书的一手参考资料，包含官方文档、论文和规范链接。

场景表示 · USD

OpenUSD: Introduction to USD
Pixar Animation Studios / AOUSD
OpenUSD 官方入门文档，涵盖 Stage / Prim / Layer / Composition 基础。对应本书第 1、2 章。
UsdPreviewSurface Proposal
Pixar Animation Studios / AOUSD
UsdPreviewSurface 参数规范，包含所有 inputs 的语义、范围和默认值。对应第 4 章。

PBR 材质 · Materials

LearnOpenGL: PBR Theory
Joey de Vries
Cook-Torrance microfacet BRDF 的实践讲解，含 GGX NDF、Schlick Fresnel、Smith-GGX G 的完整推导和 GLSL 代码。对应第 3 章。
Physically Based Shading at Disney
Brent Burley, SIGGRAPH 2012 Courses: Physically Based Shading in Film and Game Production
迪士尼 "principled" PBR 模型的原始论文，定义了 metallic-roughness 工作流的语义标准，是 glTF / UsdPreviewSurface 参数设计的直接来源。对应第 4 章。
NVIDIA MDL Documentation
NVIDIA Corporation
MDL（Material Definition Language）官方文档，包含 distribution functions、BSDF、material struct 和 SDK 接口。对应第 5 章。

几何与优化 · Geometry

Surface Simplification Using Quadric Error Metrics
Michael Garland, Paul S. Heckbert. SIGGRAPH 1997.
QEM 算法的原始论文。提出用 4×4 对称矩阵（quadric）量化顶点到关联平面的距离平方和，定义边坍缩的误差度量，实现在控制误差的前提下的快速网格简化。对应第 8 章。

资产交换 · glTF

glTF 2.0 Specification
The Khronos Group
glTF 2.0 完整规范，包含 scene / node / mesh / accessor / bufferView / material / texture 等所有结构定义，以及 PBR metallic-roughness 材质模型。对应第 9、10 章。

渲染硬件 · GPU

NVIDIA Turing Architecture Whitepaper
NVIDIA Corporation, 2018.
Turing GPU 架构白皮书，介绍 RT Core（BVH 加速光线追踪）和 Tensor Core（矩阵加速）的硬件设计。对应第 12 章。
NVIDIA Ampere GA102 Architecture Whitepaper
NVIDIA Corporation, 2020.
Ampere GA102 架构白皮书，记录第二代 RT Core 和第三代 Tensor Core 的改进数据。对应第 12 章。

仿真与具身 AI · Simulation & Embodied AI

NVIDIA Isaac Sim Documentation
NVIDIA Corporation
Isaac Sim 官方文档，涵盖 Omniverse Kit 架构、PhysX 集成、传感器 API、Python 绑定和 Replicator 合成数据框架。对应第 13 章。
VLNVerse: A Versatile Environment for Vision-and-Language Navigation
arXiv:2512.19021v1, 2024.
VLNVerse 论文，提出统一多个 VLN benchmark（R2R / CVDN / ScanQA 等）的框架，并支持 Isaac Sim 仿真场景。对应第 14 章。
