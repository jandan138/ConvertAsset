Glossary 中英术语表

本表收录全书约 40 个核心术语，按主题分组排列。每条目提供英文术语（可作为搜索关键词）和中文释义。

场景表示 · Scene Representation

中文English释义
场景描述语言USD / OpenUSDUniversal Scene Description，Pixar 开发的开放场景格式，是 Omniverse 的核心数据模型。
舞台StageUSD 合成后的完整场景视图，由一组 Layer 的 opinion 叠加而成。
原语 / 节点Prim场景图中的最小命名单元，可携带 Schema 类型（Mesh、Material 等）。
层LayerUSD 的单个磁盘文件（.usda / .usdc / .usd），携带一组 opinion，多个 Layer 叠加合成 Stage。
意见Opinion某个 Layer 对特定 Prim 属性的赋值，最终由 LIVRPS 规则裁定生效值。
组合弧Composition ArcUSD 的层次引用机制，包括 Reference、Payload、Inherit、Variant、Specialize 五种。
变体集Variant Set一组可切换的场景替代方案（如不同颜色的椅子），通过 Variant Arc 在同一 Stage 里按需激活。
有效负载Payload延迟加载的 Composition Arc，只有在明确"激活 Payload"时才会加载目标 Layer，适合大场景。

材质与外观 · Materials

中文English释义
基于物理的渲染PBRPhysically Based Rendering，遵循能量守恒和微表面理论的渲染模型。
双向反射分布函数BRDFBidirectional Reflectance Distribution Function，描述光从方向 $l$ 入射、方向 $v$ 出射时的能量分配比例。
微表面模型Microfacet Model把表面看作无数微型完美镜面的统计模型，Cook-Torrance BRDF 基于此推导。
法线分布函数NDF / DNormal Distribution Function，描述微表面法线与半程向量对齐的概率，GGX 是常用选择。
菲涅耳项Fresnel / F掠射角下反射增强的效应，Schlick 近似：$F = F_0 + (1-F_0)(1-v\cdot h)^5$。
几何项Geometry / G微表面间相互遮挡 / 阴影导致的能量损失，越粗糙越大。
金属度工作流Metallic-roughness Workflow用 metallic (0=dielectric, 1=conductor) 和 roughness 两个参数控制 PBR 材质的主流工作流。
USD 预览表面UsdPreviewSurfaceOpenUSD 官方规范的跨工具 PBR 材质模型，参数包括 diffuseColor、metallic、roughness、normal 等。
材质定义语言MDLMaterial Definition Language，NVIDIA 的声明式 + 过程式材质系统，renderer-bound，支持完整 BSDF。

几何与优化 · Geometry

中文English释义
主变量PrimvarUSD 中附属于 Prim 的数组型数据（UV、法线、颜色等），支持不同插值模式。
面变化插值Face-varying每个面的每个顶点各自独立的插值模式，允许不连续 UV（接缝），比顶点插值精度更高。
细节层次LODLevel of Detail，同一资产在不同距离下切换不同精度的几何体，降低渲染和物理计算开销。
二次误差度量QEMQuadric Error Metrics，Garland & Heckbert 1997 年提出的网格简化算法，用二次型矩阵量化边坍缩误差。
边坍缩Edge CollapseQEM 的基本操作：将一条边的两个端点合并为一个顶点，减少面数。

资产交换 · Asset Interchange

中文EnglishRelief
二进制 glTFGLBGL Transmission Format 的二进制容器，把 JSON 元数据、几何 buffer、纹理全部打包成单文件。
访问器AccessorglTF 数据模型中描述如何从 BufferView 读取具体数据类型（vec3、mat4 等）的结构。
缓冲视图BufferViewglTF 中对 Buffer 的切片引用，描述偏移量和字节长度。

渲染硬件 · Rendering Hardware

中文English释义
流式多处理器SMStreaming Multiprocessor，GPU 的基本计算单元，包含 CUDA Cores、Tensor Cores、RT Cores。
线程束WarpNVIDIA GPU 上 32 个线程组成的 SIMT 执行单元，所有线程同时执行同一条指令。
单指令多线程SIMTSingle Instruction Multiple Threads，GPU 的并行执行模型，与 CPU 的 SIMD 相似但线程数更多。
层次包围盒BVHBounding Volume Hierarchy，光线追踪的加速结构，将三角形层次组织成 AABB 树，由 RT Core 硬件加速。
实时光追核心RT CoreTuring / Ampere GPU 中专用的 BVH 遍历 + 光线三角形求交硬件单元。
张量核心Tensor Core专用于矩阵乘加运算的硬件单元，用于 AI 推理、降噪（DLSS）等任务。
深度学习超级采样DLSSDeep Learning Super Sampling，用 Tensor Core 训练的 AI 上采样网络，在低分辨率渲染后推断高分辨率图像。

仿真与 AI · Simulation & AI

中文English释义
全宇宙平台OmniverseNVIDIA 的实时 3D 协作与仿真平台，以 USD 为核心数据模型。
仿真就绪资产SimReadyNVIDIA 对仿真资产的标准：正确 pivot、物理 API、碰撞代理、合理 LOD，完全用 USD 表达。
物理引擎PhysXNVIDIA 的实时物理仿真引擎，集成在 Isaac Sim 中，通过 USD Physics Schema 接受物理属性。
合成数据Synthetic Data在仿真环境中生成的训练 / 测试数据，可大规模生产，用于减少对真实数据的依赖。
复制器ReplicatorIsaac Sim / Omniverse 的合成数据生成框架，支持随机化场景、材质、光照和相机。
视觉语言导航VLNVision-Language Navigation，根据自然语言指令在 3D 环境中自主导航到目标位置的具身 AI 任务。
视觉语言模型VLMVision-Language Model，同时处理图像和文本的大模型，具备 grounding 能力。
语言定位Grounding将自然语言指代表达映射到图像中具体空间区域（坐标或 bbox）的能力。
仿真到真实迁移Sim-to-Real在仿真环境训练的策略迁移到真实机器人时的性能差距及弥合方法。
接地房间场景GRScenesGrounded Room Scenes，为 VLN grounding 研究设计的室内场景数据集，支持配对材质渲染对比。
证据门控协议Evidence-gate ProtocolConvertAsset 论文提出的四门评测框架：视觉相似度、grounding、material-effect risk、具身栈可用性，必须独立通过。
材质效果风险Material-effect RiskMDL 近似损失对 VLM grounding 的潜在影响，按材质类型分级（低 / 中 / 高）。

评测指标 · Metrics

中文English释义
峰值信噪比PSNRPeak Signal-to-Noise Ratio，像素均方误差转换的 dB 值，越高越好。ConvertAsset 结果：35.52 dB。
结构相似度SSIMStructural Similarity Index，同时衡量亮度、对比度和结构，1 为完美。ConvertAsset 结果：0.990。
感知图像块相似度LPIPSLearned Perceptual Image Patch Similarity，用 CNN 特征层计算，0 为完全相同，越低越好。ConvertAsset 结果：0.020。
CLIP 余弦相似度CLIP SimilarityOpenAI CLIP 图像嵌入的余弦相似度，衡量语义层面的相似性，1 为完全相同。ConvertAsset 结果：0.925。
DINOv2 相似度DINOv2 SimilarityMeta DINOv2 图像嵌入的余弦相似度，对纹理细节更敏感，1 为完全相同。ConvertAsset 结果：0.872。
导航成功率SRSuccess Rate，VLN 中 agent 停在目标 $d$ 米内的 episode 比例。
路径长度加权成功率SPLSuccess weighted by Path Length，惩罚绕路的导航成功率指标。
