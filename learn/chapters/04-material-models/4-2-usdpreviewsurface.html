UsdPreviewSurface 参数详解

UsdPreviewSurface 是 Pixar 为 USD 设计的通用预览着色器（node id：UsdPreviewSurface），目标是在 DCC 工具、实时引擎和离线渲染器之间实现可靠的材质交换。它同时支持 metalness workflow 和 specular workflow，通过 useSpecularWorkflow 参数切换。ConvertAsset 的 no-MDL 转换流程正是把 MDL 材质落到这个格式。

From GAMES101 你在 GAMES101 里直接操作 shader 代码。UsdPreviewSurface 把 shader 参数抽象成了 USD 属性（attribute），可以连接纹理网络（UsdUVTexture）或直接设置标量值——本质上是同一套 Cook-Torrance PBR，只是换了个声明式的接口。

核心参数一览

以下参数在 metalness workflow（useSpecularWorkflow=0）下均有效：

参数 类型 默认值 含义
diffuseColorcolor3f(0.18, 0.18, 0.18)metalness 下为 albedo；specular 下为漫反射颜色
metallicfloat0.0金属度，0=电介质，1=导体；仅 metalness workflow
roughnessfloat0.5GGX 粗糙度（renderer 内部做 \(\alpha = r^2\)）
specularColorcolor3f(0.0, 0.0, 0.0)specular workflow 下的 F0；metalness 下忽略
iorfloat1.5折射率；metalness=0 时推导 F0；影响透明与 clearcoat
clearcoatfloat0.0第二高光叶（白色）强度，模拟清漆层
clearcoatRoughnessfloat0.01清漆层的 GGX 粗糙度
opacityfloat1.0透明度；0 = 全透明，1 = 不透明
normalnormal3f(0,0,1)切线空间法线贴图，\([-1,1]^3\)；需 scale/bias 转换
occlusionfloat1.0AO（环境遮挡）；0=完全遮挡，1=完全不遮挡

useSpecularWorkflow 切换

useSpecularWorkflow 是一个整数开关（0 或 1），只能是 connectability = "interfaceOnly"（不能连纹理）：

  • 0（默认，metalness workflow）：metallic 参数生效；F0 由 metallic 与 0.04 混合计算；specularColor 被忽略。
  • 1（specular workflow）：specularColor 直接作为 F0；metallic 被忽略。
$$ \text{F0} = \begin{cases} \text{mix}(0.04,\; \text{diffuseColor},\; \text{metallic}) & \text{useSpecularWorkflow}=0 \\ \text{specularColor} & \text{useSpecularWorkflow}=1 \end{cases} $$

pxr 代码示例

以下示例展示如何用 pxr Python 绑定创建一个带完整参数的 UsdPreviewSurface 节点：

create_preview_surface.py
from pxr import Usd, UsdShade, Sdf

stage = Usd.Stage.CreateNew("material.usda")
mat = UsdShade.Material.Define(stage, "/Looks/CarPaint")
shader = UsdShade.Shader.Define(stage, "/Looks/CarPaint/Surface")
shader.CreateIdAttr("UsdPreviewSurface")

# metalness workflow（默认）
shader.CreateInput("useSpecularWorkflow", Sdf.ValueTypeNames.Int).Set(0)

# 深红色车漆
shader.CreateInput("diffuseColor",  Sdf.ValueTypeNames.Color3f).Set((0.6, 0.05, 0.05))
shader.CreateInput("metallic",      Sdf.ValueTypeNames.Float).Set(0.0)
shader.CreateInput("roughness",     Sdf.ValueTypeNames.Float).Set(0.25)

# 清漆层（汽车漆特有）
shader.CreateInput("clearcoat",          Sdf.ValueTypeNames.Float).Set(0.9)
shader.CreateInput("clearcoatRoughness", Sdf.ValueTypeNames.Float).Set(0.02)

# 法线与 AO
shader.CreateInput("normal",    Sdf.ValueTypeNames.Normal3f).Set((0.0, 0.0, 1.0))
shader.CreateInput("occlusion", Sdf.ValueTypeNames.Float).Set(1.0)

# 连接到 Material 的 surface 输出
mat.CreateSurfaceOutput().ConnectToSource(
    shader.ConnectableAPI(), "surface"
)
specular_workflow_example.py
from pxr import UsdShade, Sdf

# specular workflow：直接指定 F0 颜色（适合金属质感的 tinted specular）
shader.CreateInput("useSpecularWorkflow", Sdf.ValueTypeNames.Int).Set(1)
shader.CreateInput("diffuseColor",  Sdf.ValueTypeNames.Color3f).Set((0.01, 0.01, 0.01))
# F0 for gold：(1.00, 0.71, 0.29) in linear
shader.CreateInput("specularColor", Sdf.ValueTypeNames.Color3f).Set((1.0, 0.71, 0.29))
shader.CreateInput("roughness",     Sdf.ValueTypeNames.Float).Set(0.15)

法线贴图的注意事项

UsdPreviewSurface 要求 normal 输入是切线空间（tangent space）法线，范围 \([-1, 1]^3\)。通过 UsdUVTexture 读取 8-bit 法线贴图时，需要设置 scale = (2, 2, 2, 1) 和 bias = (-1, -1, -1, 0) 完成 \([0,255] \to [-1,1]\) 转换。同时必须把 sourceColorSpace 设为 raw（不做 sRGB 解码），否则法线方向会错误。

ConvertAsset 的实践 ConvertAsset 在 no-MDL 转换时，把 MDL 材质的各个输入（albedo、roughness、metallic、normal 纹理）逐一映射到 UsdPreviewSurface 对应参数。no_mdl/materials.py 就是做这件事的模块。
