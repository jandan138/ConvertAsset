quadric error metric 推导

QEM 的核心洞见是：一个顶点离其周围所有原始平面有多远，就是这个顶点的几何误差。用所有平面的平方距离之和来度量误差，这个函数恰好是一个关于顶点坐标的二次型——可以用一个 4×4 矩阵（quadric Q）紧凑表示，且矩阵满足加法性（两个顶点合并 → \(\bar{Q} = Q_1 + Q_2\)）。

从平面方程到距离

三维空间里，平面方程写为：

$$ ax + by + cz + d = 0, \quad a^2 + b^2 + c^2 = 1 $$

其中 \(\mathbf{n} = (a,b,c)^T\) 是单位法线，\(d = -\mathbf{n} \cdot \mathbf{p}\)（\(\mathbf{p}\) 为面上任意一点）。一个点 \(\mathbf{v} = (v_x, v_y, v_z)^T\) 到该平面的有向距离是 \(a v_x + b v_y + c v_z + d\)；平方距离是：

$$ \Delta(\mathbf{v}) = (a v_x + b v_y + c v_z + d)^2 $$

二次型矩阵 K = pp^T

引入齐次坐标 \(\mathbf{v}^h = (v_x, v_y, v_z, 1)^T\)，则平方距离可以写成一个二次型：

$$ \Delta(\mathbf{v}) = (\mathbf{p}^T \mathbf{v}^h)^2 = (\mathbf{v}^h)^T (\mathbf{p} \mathbf{p}^T) \mathbf{v}^h $$

其中 \(\mathbf{p} = (a, b, c, d)^T\)（平面参数向量），\(K = \mathbf{p}\mathbf{p}^T\) 是一个 4×4 对称矩阵（fundamental error quadric）：

$$ K = \mathbf{p}\mathbf{p}^T = \begin{pmatrix} a^2 & ab & ac & ad \\ ab & b^2 & bc & bd \\ ac & bc & c^2 & cd \\ ad & bd & cd & d^2 \end{pmatrix} $$

顶点 quadric：累加所有相邻面

一个顶点 \(v\) 的 quadric \(Q(v)\) 定义为它所有相邻面的 \(K\) 矩阵之和：

$$ Q(v) = \sum_{\text{面} f \in \text{adj}(v)} K_f $$

这样，顶点 \(v\) 的"几何误差"（它离所有相邻原始面的总平方距离）就是：

$$ \mathcal{E}(v) = (\mathbf{v}^h)^T Q(v) \mathbf{v}^h $$

这个量越大，说明顶点离原始曲面越远，几何误差越大。

加法性：为什么可以合并

当执行坍塌 \((v_1, v_2) \to \bar{v}\) 时，新顶点 \(\bar{v}\) 的 quadric 是两个端点 quadric 的和：

$$ \bar{Q} = Q(v_1) + Q(v_2) $$

这个加法性来自矩阵的线性叠加：\(\bar{v}\) 同时继承了 \(v_1\) 和 \(v_2\) 的所有相邻面平面，所以用两矩阵之和来估计 \(\bar{v}\) 到所有这些平面的总误差，是自洽的近似。

Python 实现：plane_quadric

ConvertAsset 的 mesh/simplify.py 直接用列表实现 4×4 矩阵（行主序，16 个浮点）：

mesh/simplify.py（节选）
def plane_quadric(a, b, c, d):
    """由平面参数 [a,b,c,d] 构建 K = pp^T（4×4，行主序列表）"""
    return [
        a*a, a*b, a*c, a*d,   # 第一行
        b*a, b*b, b*c, b*d,   # 第二行
        c*a, c*b, c*c, c*d,   # 第三行
        d*a, d*b, d*c, d*d,   # 第四行
    ]

# 初始化阶段：遍历每个三角面，累加到三个顶点
for fi, f in enumerate(F):
    i, j, k = f
    n = cross(sub(V[j], V[i]), sub(V[k], V[i]))  # 面法线
    norm = length(n)
    n = [n[0]/norm, n[1]/norm, n[2]/norm]         # 归一化
    d = -dot(n, V[i])                              # 平面常数项
    K = plane_quadric(n[0], n[1], n[2], d)
    add_inplace(v_quads[i], K)   # Q(v_i) += K
    add_inplace(v_quads[j], K)   # Q(v_j) += K
    add_inplace(v_quads[k], K)   # Q(v_k) += K
