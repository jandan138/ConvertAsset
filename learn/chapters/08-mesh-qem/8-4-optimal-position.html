最优 contraction 位置求解

给定合并后的 quadric \(\bar{Q} = Q_1 + Q_2\)，我们要找到一个点 \(\bar{v}\) 使得误差 \(\mathcal{E}(\bar{v}) = (\bar{v}^h)^T \bar{Q} \bar{v}^h\) 最小。因为 \(\mathcal{E}\) 是 \(\bar{v}\) 坐标的二次函数，最小值点满足梯度为零——这等价于解一个 4×4（实际 3×3）线性方程组。当方程组病态时，退回到端点中点作为安全回退。

对误差函数求梯度

把 \(\bar{v}^h = (x, y, z, 1)^T\) 代入 \(\mathcal{E} = (\bar{v}^h)^T \bar{Q} \bar{v}^h\)，对 \(x, y, z\) 分别求偏导并令其等于零：

$$ \frac{\partial \mathcal{E}}{\partial x} = \frac{\partial \mathcal{E}}{\partial y} = \frac{\partial \mathcal{E}}{\partial z} = 0 $$

这等价于（利用 \(\bar{Q}\) 的对称性）：

$$ \begin{pmatrix} q_{11} & q_{12} & q_{13} \\ q_{12} & q_{22} & q_{23} \\ q_{13} & q_{23} & q_{33} \end{pmatrix} \begin{pmatrix} x \\ y \\ z \end{pmatrix} = -\begin{pmatrix} q_{14} \\ q_{24} \\ q_{34} \end{pmatrix} $$

即 \(A \bar{x} = b\)，其中 \(A\) 是 \(\bar{Q}\) 的左上 3×3 子矩阵，\(b = -(q_{14}, q_{24}, q_{34})^T\)。注意第 4 行等式 \(0 \cdot x + 0 \cdot y + 0 \cdot z + 1 = 1\) 恒成立，因此只需解 3×3 系统。

高斯消元（带部分选主元）

ConvertAsset 的 mesh/simplify.py 用带部分选主元（partial pivoting）的高斯消元法求解这个 3×3 系统，当主元 \(< 10^{-12}\) 时判定矩阵奇异并返回 None：

mesh/simplify.py（solve3 + optimal_position_cost）
def optimal_position_cost(Q, pu, pv):
    """求合并代价最小的位置（解 3x3 方程组或回退到中点）"""
    A = [[Q[0],Q[1],Q[2]],
         [Q[4],Q[5],Q[6]],
         [Q[8],Q[9],Q[10]]]
    b = [-Q[3], -Q[7], -Q[11]]
    x = solve3(A, b)          # 高斯消元
    if x is None:             # 矩阵病态 → 退化到端点中点
        x = [(pu[0]+pv[0])*0.5,
             (pu[1]+pv[1])*0.5,
             (pu[2]+pv[2])*0.5]
    vx, vy, vz = x
    v4 = [vx, vy, vz, 1.0]
    cost = quadric_eval(Q, v4)   # v^T Q v
    return (vx, vy, vz), float(cost)

def quadric_eval(Q, v4):
    """计算 v^T Q v（4×4 Q，4-vector v）"""
    Qv = [sum(Q[i*4+j]*v4[j] for j in range(4)) for i in range(4)]
    return sum(v4[i]*Qv[i] for i in range(4))

回退策略：候选位置的比较

当 \(A\) 奇异时，Garland & Heckbert 建议在三个候选位置（\(v_1\)、\(v_2\)、中点）中选代价最小的。ConvertAsset 简化为直接用中点——这是一个工程取舍：实现简单，对绝大多数普通网格效果差异可忽略。

坍塌代价的直观理解

坍塌代价 \(\mathcal{E}(\bar{v}) = (\bar{v}^h)^T \bar{Q} \bar{v}^h\) 的量纲是"平方距离之和"——近似于新顶点离原始曲面所有相邻平面的平均平方距离。代价越大，坍塌越会引起视觉上可见的误差。优先队列每次弹出代价最小的坍塌，就是贪心地保留几何误差最小的操作。

小结 最优坍塌位置 = 解 3×3 线性方程组（来自对 \(v^T Q v\) 求偏导置零）。实现要点：带部分选主元的高斯消元 + 病态时中点回退。ConvertAsset 的 solve3 函数完整实现了这个过程，是整个 QEM 流程里最"数学"的部分。
