论文的 evidence-gate protocol

视觉相似度指标（PSNR、SSIM、LPIPS 等）再高，也只说明"图像像"，而不能证明"下游任务不受影响"。ConvertAsset 论文因此提出 evidence-gate protocol（证据门控协议）：视觉相似度、grounding、material-effect risk、具身栈可用性这四个 gate 必须作为独立证据分别报告，不能用其中任何一个替代其他三个。

为什么需要 evidence gate

传统的转换质量评测只看像素相似度（PSNR / SSIM），默认"像素相似 → 下游任务等价"。这个假设在以下情形下会失效：

  • VLM 对颜色语义非常敏感（如"红色沙发"），即使 PSNR 很高，如果转换把暗红色变成了偏棕色，grounding 结果可能改变。
  • 材质类型（如 clearcoat、procedural texture）的语义可能对 VLM 的物体识别有影响，但 SSIM 感知不到。
  • 具身导航的路径长度和成功率取决于场景的可导航性（navigation feasibility），而不仅仅是视觉外观。

论文直接从 abstract 提出："proxy fidelity alone is insufficient for downstream grounding claims"（代理保真度不足以支撑下游 grounding 主张）。四个 gate 的设计正是将这一认识形式化为可操作的评测清单。

四个 gate 的定义如下：

Gate 1 · 视觉相似度（Visual Similarity）

衡量转换后渲染与原始渲染的像素和特征相似度。ConvertAsset 的结果：PSNR 35.52 dB、SSIM 0.990、LPIPS 0.020、CLIP 0.925、DINOv2 0.872——此 gate 通过。但注意，通过 Gate 1 只是必要条件，不是充分条件。

Gate 2 · Grounding（语言定位）

在 GRScenes 配对渲染上，评测 VLM 的目标类别识别和坐标定位能力。ConvertAsset 当前状态：pilot 阶段（15 对 clean pool + 30 对 stress set），尚未达到 final benchmark 规模（final_benchmark_claimable=false）。Gemma4 在两种材质条件下答案一致性高，Qwen2.5-VL 暴露了坐标协议敏感性。

Gate 3 · Material-effect risk（材质效果风险）

针对材质类型逐一评估转换的近似损失和对 grounding 的潜在影响。ConvertAsset 的 risk matrix 涵盖六个 bin（见 14-5 节），其中 clearcoat 和 procedural texture 是需要人工审核的风险类型。

Gate 4 · 具身栈可用性（Embodied-stack Usability）

在真实 VLN 评测平台（InternNav / InteriorAgent，官方 KuJiaLe 场景）上运行完整 episode。ConvertAsset 的结果：99 episode val_unseen sanity run，noMDL 与 original 的 SR / SPL / NE / TL 差异在统计误差范围内（paired bootstrap CI 与 0 重叠）——此 gate 通过（在当前测试范围内）。

Gate 状态汇总 Gate 1 视觉相似度：✓ 通过（高 PSNR / SSIM / CLIP）
Gate 2 Grounding：⚠ pilot 阶段，未最终定论
Gate 3 Material-effect risk：⚠ 部分材质类型需人工审核
Gate 4 具身栈：✓ 通过（官方场景 SR/SPL 统计等价）

claim 边界矩阵

evidence-gate protocol 的另一个设计目标是防止过度 claim。论文明确区分了：

  • 可以主张（claimable）："在当前 4 件资产 / 6 视角的范围内，视觉代理保真度高"——已有量化证据。
  • 不可主张：用视觉相似度推断"对所有具身任务无影响"——超出现有证据范围。
  • 需要额外证据：grounding accuracy 和 navigation success rate 的全面对比——还在 pilot 阶段。

这种"claim 边界意识"是科学严谨性的体现：我们知道自己测了什么，也知道自己没有测什么。
