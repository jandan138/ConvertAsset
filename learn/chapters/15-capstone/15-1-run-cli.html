ConvertAsset Learning Guide
PART VII · 综合 · Capstone

端到端跑一遍 CLI

这一节把整本书学过的概念串联成一个可以真正运行的操作序列。从一个带 MDL 材质的原始 USD 资产出发，依次经过材质转换、网格简化、GLB 导出，最后用本地 Isaac Sim 出图——每一步都有对应的 ConvertAsset CLI 子命令。所有命令都通过 ./scripts/isaac_python.sh 在 Isaac Sim 的 Python 环境下运行。

Step 0：准备环境

确认 Isaac Sim 已安装，并设置好路径变量：

环境检查
# 检查 Isaac Sim Python 脚本是否可执行
ls -la ./scripts/isaac_python.sh

# （可选）显式指定 Isaac Sim 安装路径
export ISAAC_SIM_ROOT=/isaac-sim

# 确认 pxr 可导入
./scripts/isaac_python.sh -c "from pxr import Usd; print('pxr ok:', Usd.__version__)"

./scripts/isaac_python.sh 会自动检测 Isaac Sim 的安装位置（通过 $ISAAC_SIM_ROOT 环境变量或默认路径），设置 PYTHONPATH 和 LD_LIBRARY_PATH，然后执行 Python 脚本。

Step 1–4：逐步运行

Step 1：no-mdl — MDL → UsdPreviewSurface

这是 ConvertAsset 的核心功能（第 6 章），递归处理 USD 及其所有依赖，为每个文件生成 *_noMDL.usd 兄弟文件：

1_no_mdl.sh
./scripts/isaac_python.sh ./main.py no-mdl \
    /data/assets/chair/chair.usd

# 成功输出示例：
# === SUMMARY ===
# /data/assets/chair/materials/wood.usd -> /data/assets/chair/materials/wood_noMDL.usd
# /data/assets/chair/chair.usd -> /data/assets/chair/chair_noMDL.usd
# Top-level new file: /data/assets/chair/chair_noMDL.usd

关键选项：--only-new-usd 跳过生成旁路 summary 文件（加速批处理）；--resolve-textures-to-absolute 把纹理路径写为绝对路径（兼容旧行为）。

Step 2：mesh-faces — 检查原始面数

在简化前先检查网格面数，确认有没有简化的必要：

2_check_faces.sh
./scripts/isaac_python.sh ./main.py mesh-faces \
    /data/assets/chair/chair_noMDL.usd

# 输出：Total render mesh faces: 847392

Step 3：mesh-simplify — QEM 网格简化

把面数缩减到 50%，用纯 Python backend（无需编译 C++）：

3_simplify.sh
./scripts/isaac_python.sh ./main.py mesh-simplify \
    /data/assets/chair/chair_noMDL.usd \
    --ratio 0.5 \
    --backend py \
    --apply \
    --out /data/assets/chair/chair_noMDL_simplified.usd \
    --progress

# 输出：
# [APPLY] meshes total=12 tri=12 skipped_non_tri=0
# faces: 847392 -> 423696
# verts: 423847 -> 212073
# Exported: /data/assets/chair/chair_noMDL_simplified.usd

如果已编译 C++ backend，可以用 --backend cpp-uv 获得更快的处理速度（带 UV 保持）。

Step 4：export-glb — USD → GLB

把简化后的 noMDL USD 导出为标准 GLB 文件（第 9、10 章的流水线）：

4_export_glb.sh
./scripts/isaac_python.sh ./main.py export-glb \
    /data/assets/chair/chair_noMDL_simplified.usd \
    --out /data/assets/chair/chair.glb

# 或者一步到位（no-mdl + export-glb + 清理中间文件）：
./scripts/isaac_python.sh ./main.py usd-to-glb \
    /data/assets/chair/chair.usd \
    --out /data/assets/chair/chair.glb

输出验证

完成后，用 inspect 子命令验证转换结果：

5_inspect.sh
# 查看某个材质的 UsdPreviewSurface 网络（替换为实际材质路径）
./scripts/isaac_python.sh ./main.py inspect \
    /data/assets/chair/chair_noMDL.usd \
    usdpreview \
    /Looks/Material_Wood

# 查看 GLB 文件大小，确认成功生成
ls -lh /data/assets/chair/chair.glb
完整流程一览 no-mdl → mesh-faces（可选检查）→ mesh-simplify → export-glb
或者一步到位：usd-to-glb（自动执行 no-mdl + export-glb + 清理中间文件）
