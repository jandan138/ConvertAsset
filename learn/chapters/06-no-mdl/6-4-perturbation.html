ConvertAsset Learning Guide
PART II · 材质与外观 · Materials & Appearance

论文视角：conversion as controlled perturbation

从工程角度，no-MDL 转换是一个"让资产变得可移植"的基础设施步骤，看似无关紧要。ConvertAsset 论文（投稿 ACL27）换了一个研究角度：把这个转换视为一种受控的仿真内扰动（controlled within-simulation perturbation），系统地研究它对下游 language grounding 任务的影响。高 proxy 相似度（PSNR / SSIM 等）不等于下游任务无影响——这是论文的核心主张。

问题框架：perturbation 视角

论文的出发点是：合成 3D benchmark 越来越多地用渲染的 USD 场景来评测视觉语言和具身智能 Agent，而这些场景在发布时往往需要做材质转换（MDL → UsdPreviewSurface）以满足 portability 需求。这个转换通常被当作中性的基础设施操作（neutral infrastructure），不去研究它是否改变了评测结论。

ConvertAsset 论文的贡献在于：对这个"中性假设"提出质疑，把转换建模为一种扰动，并设计多层证据门（evidence gates）来量化其影响。

Proxy 相似度指标

论文对四个 Isaac Sim 资产测量了转换前后的渲染图像相似度（同场景、同光照、同相机），proxy 指标如下：

指标 值 含义
PSNR 35.52 dB 像素级均方误差；>30 dB 通常认为图像质量"好"
SSIM 0.990 结构相似度；接近 1.0 表示结构高度一致
LPIPS 0.020 感知相似度（越低越好）；0.02 极小
CLIP cosine 0.925 语义特征余弦相似度；高值说明语义表征接近
DINOv2 cosine 0.872 视觉特征相似度；中高水平

这些数字乍看非常好——PSNR 35 dB，SSIM 0.99，图像几乎看不出差异。这意味着什么？

Proxy 指标的局限：充分条件 vs 必要条件

论文核心主张 proxy fidelity alone is insufficient for downstream grounding claims. 高 proxy 相似度是必要条件（similarity is necessary），但不是充分条件（not sufficient）。即使 PSNR 35 dB，材质转换仍然可能改变：光泽高光的位置与亮度、金属质感的感知（金属 vs 非金属误判）、颜色的轻微偏移——这些细微变化可能在特定 grounding 问题下影响 VLM 的回答。

论文的实验设计因此分为多个证据门：

  1. 视觉相似度门：PSNR / SSIM / LPIPS / CLIP / DINOv2（proxy 指标，已通过）。
  2. Language grounding 门：用 Gemma4 和 Qwen2.5-VL 在 GRScenes 数据集上做 15-pair 视觉 QA，对比原始场景与转换后场景的答对率。实验结果：Gemma4 在两种条件下均保持 30/30 target category 答对；Qwen2.5-VL 暴露出在坐标系下的偏好差异（raw-coordinate preference）。
  3. Material-effect 风险门：针对材质本身作为 grounding target 的场景，额外进行 NVIDIA baseline 材质影响审计。
  4. Embodied stack 可用性门：99-episode 的 InternNav sanity run，检查转换后的场景是否影响导航任务的可用性。
为什么需要 composition-preserving 转换 论文能够做"配对实验"（原始 vs 转换后完全相同的场景结构、相机位置、光照），正是因为 ConvertAsset 的非打平设计保留了 USD 组合结构——只换材质，不改几何、不改层次、不改变体。这让对照实验有效，排除了其他变量干扰。

Evidence-gate protocol

论文提出的 evidence-gate protocol 是一个通用框架：任何将合成场景材质转换用于下游任务的工作，都应该分别报告以下四个维度的证据，而不是用单一的 proxy 指标代替全部：

  • 视觉相似度（PSNR / SSIM / LPIPS / CLIP / DINOv2）
  • Grounding 性能（VQA / 标注一致性）
  • 材质影响风险（当材质本身是任务的语义目标时）
  • Embodied stack 可用性（导航 / 操作任务的功能完整性）
小结 no-MDL 转换在 proxy 指标上表现优秀（PSNR 35.52 dB，SSIM 0.990，LPIPS 0.020），但这不足以保证下游任务不受影响。论文的贡献是给出了一套有界（claim-bounded）的评测协议，让"材质转换对 VLM 评测有多大影响"这个问题第一次得到系统回答。
