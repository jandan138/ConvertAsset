算法：MDL 网络 → UsdPreviewSurface

no-MDL 转换的核心是一套"读取—映射—构建—清理"的四步算法，实现在 convert_asset/no_mdl/materials.py 与 convert.py 中。它不依赖 MDL SDK，而是直接在 USD Shader prim 的 inputs: 上读取参数，配合可选的 .mdl 文件文本解析，然后手工搭建 UsdPreviewSurface 网络。

第一步：定位 MDL Shader

find_mdl_shader(mat) 函数在一个 Material Prim 下遍历所有子 Prim，找到 info:implementationSource = "sourceAsset" 且 info:mdl:sourceAsset 存在的 Shader prim。这就是 MDL 着色器的位置。

查找逻辑优先 MATERIAL_ROOT_HINTS = ("/Root/Looks", "/World/Looks") 指定的路径前缀，找不到才扫描整个 Stage（ALWAYS_SCAN_ALL_MATERIALS = True）。

第二步：提取参数

参数提取是两条并行路径，取最终有效值：

  1. USD inputs 直读——直接访问 Shader prim 的 inputs:diffuse_color_constant、inputs:reflection_roughness_constant、inputs:metallic_constant 等字段。这对 OmniPBR / OmniSurface 系 MDL 效果最好，因为它们有规范的 input 命名。
  2. MDL 文件文本解析（mdl_parse.py）——当 USD inputs 读不到值时，回退到解析 .mdl 源文件文本。parse_mdl_text() 用正则表达式提取 diffuse: color(r,g,b)、roughness: 0.4、metallic: 0.0 等 key-value 形式，以及文件中引用的贴图路径（匹配 .png、.exr 等扩展名的字符串字面量）。

贴图提取由 copy_textures() 完成，它通过 USD input 连接关系（如 inputs:diffuse_texture.connect = ...）或 MDL 文本中的文件名启发式匹配来确定 BaseColor / Roughness / Metallic / Normal 四个通道对应的贴图路径。

第三步：构建 UsdPreviewSurface 网络

ensure_preview(stage, mat) 在 Material 下方建立固定布局的 PreviewSurface 网络：

ensure_preview 构建的网络结构（伪代码）
def ensure_preview(stage, mat):
    # 在 /Looks/MyMat/PreviewNetwork/ 下建立 Scope
    scope = stage.DefinePrim(f"{mat_path}/PreviewNetwork", "Scope")

    # PreviewSurface 节点
    prev = UsdShade.Shader.Define(stage, f"{scope_path}/PreviewSurface")
    prev.CreateIdAttr("UsdPreviewSurface")

    # PrimvarReader —— 读取 UV set "st"
    uvr = UsdShade.Shader.Define(stage, f"{scope_path}/Primvar_st")
    uvr.CreateIdAttr("UsdPrimvarReader_float2")
    uvr.CreateInput("varname", Sdf.ValueTypeNames.Token).Set("st")

    # 四个 UsdUVTexture 节点（BaseColor / Roughness / Metallic / Normal）
    for tag in ("BaseColor", "Roughness", "Metallic", "Normal"):
        tex = UsdShade.Shader.Define(stage, f"{scope_path}/Tex_{tag}")
        tex.CreateIdAttr("UsdUVTexture")
        tex.CreateInput("st", Sdf.ValueTypeNames.Float2).ConnectToSource(uv_out)
        tex.CreateInput("wrapS", Sdf.ValueTypeNames.Token).Set("repeat")
        tex.CreateInput("wrapT", Sdf.ValueTypeNames.Token).Set("repeat")

    # Material 的 outputs:surface 连接到 PreviewSurface.outputs:surface
    mat.CreateSurfaceOutput().ConnectToSource(prev_api, "surface")

随后 connect_preview() 把提取到的贴图路径写入对应 UsdUVTexture 节点的 inputs:file，并建立连接；如果某通道无贴图，则直接在 PreviewSurface 上写入常量值（BaseColor 默认 0.18 灰，Roughness 默认 0.5，Metallic 默认 0.0）。

Roughness / Gloss 互转 部分 MDL 材质（如某些布料类）用 glossiness（光泽度 = 1 − roughness）来表达粗糙度。materials.py 检测到 gloss 贴图后，会在 UsdUVTexture 上设置 inputs:scale = (−1, −1, −1, −1) 和 inputs:bias = (1, 1, 1, 1)，实现反转，直接连到 PreviewSurface.inputs:roughness。

第四步：移除 MDL 残留

构建好 PreviewSurface 之后，需要清理所有 MDL 相关节点，否则验证失败：

  • remove_material_mdl_outputs(stage)——遍历所有 Material Prim，删除 outputs:mdl:surface、outputs:mdl:volume 等 MDL 连接端口（用 Usd.Attribute.Block() 断开连接）。
  • remove_all_mdl_shaders(stage)——删除所有 info:implementationSource = "sourceAsset" 且带 info:mdl: 前缀的 Shader prim（stage.RemovePrim()）。
  • 最终由 verify_no_mdl(stage) 检查是否有残留。
外部引用的 MDL（External MDL） 如果 MDL Shader 定义在被引用的子 USD 文件里（不在当前 root layer），直接 RemovePrim 无效——那个 Shader prim 是由外部文件"贡献"的，在当前 layer 操作不到。这时 ConvertAsset 会：① 对外部 MDL Shader 调用 SetActive(False) 使其失活（DEACTIVATE_EXTERNAL_MDL_SHADERS = True）；② 额外一轮 RemovePrim 扫描（CLEAN_DELETE_EXTERNAL_MDL_SHADERS = True）；③ 若配置允许，视为"external-only-pass"通过（ALLOW_EXTERNAL_ONLY_PASS = True）。
convert.py — 主转换入口
from pxr import Usd, UsdShade

def convert_and_strip_mdl_in_this_file_only(stage: Usd.Stage) -> dict:
    """对 stage（已是 *_noMDL.usd 的副本）执行材质转换。"""
    from .materials import (
        find_mdl_shader, ensure_preview, copy_textures,
        connect_preview, remove_material_mdl_outputs,
        remove_all_mdl_shaders
    )
    stats = {"total": 0, "preview": 0}

    for prim in stage.Traverse():
        if not prim.IsA(UsdShade.Material):
            continue
        mat = UsdShade.Material(prim)
        stats["total"] += 1
        mdl_shader = find_mdl_shader(mat)
        if not mdl_shader:
            continue

        # 搭建骨架
        preview = ensure_preview(stage, mat)
        # 提取 & 连接贴图/常量
        filled, has_c, c_rgb, bc_tex = copy_textures(stage, mdl_shader, mat)
        connect_preview(stage, mat, filled, has_c, c_rgb, bc_tex)
        stats["preview"] += 1

    # 统一清理 MDL outputs 和 Shader prims
    remove_material_mdl_outputs(stage)
    remove_all_mdl_shaders(stage)
    return stats
