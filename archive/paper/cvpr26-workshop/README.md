# CVPR 2026 Workshop Archive

This directory is a PDF-only snapshot of the original CVPR 2026 workshop draft.
`main.pdf` is the preserved historical artifact; `main.tex` is retained only as
source-context metadata from the pre-migration layout and is not intended to be
rebuilt in place.

Use `paper/venues/cvpr26` for the rebuildable historical baseline under the
current multi-venue paper tree.
